# Ryu 文件

歡迎使用 Ryu 的完整文件。Ryu 是一個 AI 驅動的試算表自動化平台，讓您能夠使用 AI 代理、結構化資料模型和外部 API 整合來建立智慧自動化工作流程。

## 什麼是 Ryu？

Ryu 是一個強大的無程式碼/低程式碼解決方案，結合了：
- **AI 驅動的代理** - 具備推理和生成能力的智慧自動化
- **結構化資料管理** - 類似資料庫的表單，用於組織資訊
- **動態表單** - 友善的使用者介面，用於資料收集
- **智慧動作** - AI 驅動的多步驟工作流程處理
- **API 整合** - 透過 OAuth 連接外部服務
- **自動化排程** - 在您的資料上按排程執行動作

## 功能一覽

### 🤖 智慧代理
建立多個代理，每個都有自己的工作區、模型和自動化工作流程。

### 📊 靈活的資料模型
使用類似資料庫的表單建構您的資料，具有可自訂的欄位和無限記錄。

### 📝 動態表單
建立供人工輸入資料的表單，自動在您的表單中建立記錄。

### ⚡ 強大的動作
設定可以執行以下操作的多步驟 AI 動作：
- 執行 AI 推理和分析
- 使用 AI 生成圖像
- 執行外部服務的 API 呼叫
- 將結果儲存回您的表單

### 🔗 API 整合
透過 OAuth 認證連接外部服務（Facebook、Threads 等）。每個工作區每種服務類型只能有一個連接。

### ⏰ 智慧排程
設定自動排程來處理整個表格，在指定的時間間隔對每一行執行動作。

## 系統架構

```
代理
  └── 工作區（多個）
       ├── 模型（多個）
       │    ├── 表單（欄位 + 記錄）
       │    ├── 表單（多個）
       │    └── 動作（多個）
       │         └── 步驟（多個）
       │              ├── AI 推理
       │              ├── AI 圖像生成
       │              └── API 執行
       └── API 連接（每種類型一個）
            └── OAuth 設定
```
